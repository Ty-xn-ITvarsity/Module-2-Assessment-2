# Pine City Mall Website (ITVarsity Project)

A multi-page mall website built with HTML, CSS, and JavaScript for the ITVarsity project.

## Project Overview

This project presents Pine City Mall information, including:

- Home and mall introduction
- About page and facilities
- Store directory pages
- Movie listings and showtimes
- Contact page with form and map

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## Key Pages

- `index.html` – Home
- `about.html` – About and mall highlights
- `map.html` – Mall map
- `shops.html` – Shop directory
- `movies.html` – Movie listings and booking actions
- `contact.html` – Contact details, form, and map embed

## Features

- Consistent navigation/header/footer across pages
- Responsive layout and card-based sections
- Store profile pages (fashion, food, services, etc.)
- Movie showtime selection and booking alert flow
- Trailer and booking button interactions
- Font Awesome social/media icons

## JavaScript Functionality

Main scripting is in `script.js`, including movie ticket booking logic used by the Movies page.

## How to Run

1. Download or clone the project.
2. Open the project folder in VS Code.
3. Open `index.html` in your browser (or use Live Server).

## Project Structure (Summary)

- Root HTML pages for each section and store
- `styles.css` for shared styling
- `script.js` for interactive behavior
- `images/` for logos, posters, and gallery assets

## Notes

- Ensure all image paths remain correct when moving files.
- Keep file naming consistent (including spaces/apostrophes) when linking pages.
- External icon library is loaded via Font Awesome CDN.
